# Session Reset

Usa este flujo cuando la conversacion se vuelva larga o difusa.

1. Resume el trabajo actual con `.ai/prompts/summary-refresh.md`.
2. Abre una conversacion nueva.
3. Pega `SUMMARY`.
4. Añade solo el siguiente bloque minimo de codigo o logs.
5. Pide una sola cosa:
   - analisis
   - plan
   - ejecucion de un paso

Regla:

Si vas a pegar mas de lo necesario para responder la siguiente accion, no has reducido suficiente el contexto.
